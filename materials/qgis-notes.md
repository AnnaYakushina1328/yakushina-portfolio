# Заметки по QGIS и обработке геоданных

Раздел предназначен для учебных и проектных заметок по работе с геоинформационными системами.

## Темы для пополнения

- базовые понятия: слой, атрибут, координатная система, растровые и векторные данные;
- импорт и подготовка картографических материалов;
- классификация данных по диапазонам значений;
- оформление результата на карте;
- экспорт результата и подготовка пояснений.

Этот файл можно заменить на полноценный конспект или серию коротких статей.
