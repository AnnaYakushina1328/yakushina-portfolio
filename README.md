# Личный сайт с образовательными и проектными материалами

Статический сайт для публикации на GitHub Pages.

## Структура

- `index.html` — главная страница.
- `vkr.html` — отдельная страница дипломного проекта.
- `activities.html` — страница с проектной активностью, хакатонами и наставничеством.
- `styles.css` — оформление сайта.
- `materials/` — папка для открытых материалов, заметок, конспектов и ссылок.

## Что заменить перед публикацией

3. В разделе `materials/` заменить черновые markdown-файлы на реальные материалы.
4. Добавить скриншоты проекта, презентацию или ссылку на репозиторий, если они готовы к публикации.
5. Для хакатонов можно добавить ссылки на страницы мероприятий или подтверждающие материалы, если их можно публиковать.
