# Личный сайт с образовательными и проектными материалами

Статический сайт для публикации на GitHub Pages.

## Структура

- `index.html` — главная страница.
- `vkr.html` — отдельная страница дипломного проекта.
- `activities.html` — страница с проектной активностью, хакатонами и наставничеством.
- `styles.css` — оформление сайта.
- `materials/` — папка для открытых материалов, заметок, конспектов и ссылок.
